# MotorPHPayroll

## Milestone 1: GUI Development and Interface Refinement

This project is a Java Swing-based GUI application developed for the MotorPH Employee App as part of MO-IT103 – Computer Programming 2.

The application converts the previous console-based payroll system into a graphical user interface (GUI) with proper event handling, validation, and exception handling.

## Features
- Employee Number input field
- Employee Name input field
- Pay Coverage input field
- Process Payroll button
- Clear button
- Input validation
- Error and confirmation messages
- Java Swing GUI interface

## Technologies Used
- Java
- Java Swing
- Apache NetBeans IDE
